2. RSI、MACD、布林带的计算方法和作用

指标计算方法作用
RSIRSI = 100 - 100 / (1 + RS)，RS 为 N 日平均上涨幅度除以平均下跌幅度，常用 N=14。衡量超买超卖。RSI 较高可能偏超买，较低可能偏超卖。
MACDDIF = EMA(12) - EMA(26)，DEA = EMA(DIF, 9)，MACD = 2 × (DIF - DEA)。观察趋势强弱和动量变化，常用金叉、死叉辅助判断。
布林带中轨 = N 日均线，上轨 = 中轨 + 2 × 标准差，下轨 = 中轨 - 2 × 标准差，常用 N=20。判断波动区间、压力支撑和价格偏离程度。

3. Python 加载数据、计算指标并可视化

task2 Python 代码
import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_csv("task1_daily_data.csv")
df = df.sort_values("trade_date")
close = df["close"]

delta = close.diff()
gain = delta.clip(lower=0).rolling(14).mean()
loss = (-delta.clip(upper=0)).rolling(14).mean()
df["RSI"] = 100 - 100 / (1 + gain / loss)

ema12 = close.ewm(span=12, adjust=False).mean()
ema26 = close.ewm(span=26, adjust=False).mean()
df["DIF"] = ema12 - ema26
df["DEA"] = df["DIF"].ewm(span=9, adjust=False).mean()
df["MACD"] = 2 * (df["DIF"] - df["DEA"])

df["BOLL_MID"] = close.rolling(20).mean()
df["BOLL_STD"] = close.rolling(20).std()
df["BOLL_UPPER"] = df["BOLL_MID"] + 2 * df["BOLL_STD"]
df["BOLL_LOWER"] = df["BOLL_MID"] - 2 * df["BOLL_STD"]

direction = close.diff().apply(lambda x: 1 if x > 0 else (-1 if x < 0 else 0))
df["OBV"] = (direction * df["vol"]).fillna(0).cumsum()

print(df.isna().sum())
print(df.describe())
df[["close", "BOLL_UPPER", "BOLL_MID", "BOLL_LOWER"]].plot(figsize=(12, 5), title="Bollinger Bands")
plt.show()
df[["RSI"]].plot(figsize=(12, 3), title="RSI")
plt.show()
df[["DIF", "DEA", "MACD"]].plot(figsize=(12, 4), title="MACD")
plt.show()
df[["OBV"]].plot(figsize=(12, 3), title="OBV")
plt.show()

4. 扩展指标：OBV 能量潮

OBV 根据价格涨跌方向累计成交量：收盘价上涨则加成交量，下跌则减成交量，持平则不变。它常用于观察成交量是否支持价格趋势。
